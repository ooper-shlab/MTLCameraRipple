# MTLCameraRipple

An experimental migration of GLCameraRipple to Metal.

### BUILD REQUIREMENTS:

Xcode 10.2.1, iOS SDK 12.2

(You cannot build Metal apps targeting simulators.)

### RUNTIME REQUIREMENTS:

iOS 9.0 or later, Metal capable devide
